# Быстрый старт

## Способ 1: Автоматический запуск (Windows)

Просто запустите файл `run.bat` двойным кликом. Скрипт автоматически:
1. Проверит наличие Python
2. Установит все зависимости
3. Запустит программу

## Способ 2: Ручной запуск

### Шаг 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 2. Запуск программы

```bash
python main.py
```

## Что делает программа?

1. **Загружает данные** Wine Quality с UCI ML Repository и сохраняет их локально
2. **Обучает модели**:
   - Классификация: Gradient Boosting, Random Forest, Logistic Regression, Neural Network
   - Регрессия: Gradient Boosting, Random Forest, Neural Network
3. **Генерирует объяснения**:
   - Feature Importance
   - Permutation Importance
   - Partial Dependence Plots
   - SHAP Values (Summary, Waterfall, Force plots)
4. **Создает отчет** с результатами и метриками
5. **Сохраняет все графики** в PNG формате

## Результаты

После выполнения программы будут созданы директории:

- `data/` - загруженные и обработанные данные
- `results/` - текстовые отчеты и метрики
- `plots/` - все графики в PNG
- `models/` - обученные модели

## Основные файлы результатов

- `results/LAB3_REPORT.txt` - **полный отчет о работе**
- `results/model_metrics.csv` - метрики всех моделей в CSV
- `results/dataset_info.txt` - информация о датасете

## Время выполнения

Примерное время выполнения всей программы:
- Загрузка данных: ~10 секунд
- Обучение моделей: ~2-5 минут
- Генерация объяснений: ~5-10 минут
- **Общее время: ~10-20 минут**

## Повторный запуск

При повторном запуске программа:
- **НЕ будет** заново скачивать данные (использует сохраненные)
- **БУДЕТ** заново обучать модели и генерировать объяснения

Чтобы принудительно перезагрузить данные, удалите файлы из директории `data/`.

## Настройка параметров

Все параметры можно изменить в файле `config.py`:
- Параметры моделей
- Размер SHAP выборки
- Количество индивидуальных объяснений
- Настройки визуализации

## Решение проблем

### Ошибка: "ModuleNotFoundError"
Решение: Установите зависимости
```bash
pip install -r requirements.txt
```

### Ошибка при установке SHAP
Решение: Обновите pip и setuptools
```bash
pip install --upgrade pip setuptools wheel
pip install shap
```

### Ошибка при установке TensorFlow
Решение: Установите совместимую версию
```bash
pip install tensorflow==2.13.0
```

### Программа работает очень долго
Это нормально! Вычисление SHAP values для нейронных сетей может занять 10-15 минут.
Чтобы ускорить, уменьшите параметр `SHAP_PARAMS['sample_size']` в `config.py`.
