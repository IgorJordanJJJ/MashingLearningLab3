ОСНОВНЫЕ ФАЙЛЫ ПРОГРАММЫ
========================

1. config.py
   - Конфигурационный файл со всеми настройками проекта
   - Содержит параметры моделей, настройки SHAP, PDP, визуализации
   - Можно изменять параметры без изменения кода

2. data_loader.py
   - Модуль для загрузки и подготовки данных
   - Скачивает Wine Quality Dataset с UCI ML Repository
   - Сохраняет данные локально (для повторного использования без API)
   - Подготавливает данные для классификации и регрессии

3. model_trainer.py
   - Модуль для обучения моделей
   - Классы: ClassificationTrainer, RegressionTrainer
   - Обучает модели: Gradient Boosting, Random Forest, Logistic Regression, Neural Network
   - Вычисляет метрики качества

4. model_explainer.py
   - Модуль для получения объяснений моделей
   - Методы:
     * Feature Importance (IG для деревьев, коэффициенты для регрессии)
     * Permutation Importance
     * Partial Dependence Plots (PDP)
     * SHAP Values (TreeExplainer, LinearExplainer, KernelExplainer)
   - Визуализация всех объяснений

5. report_generator.py
   - Модуль для генерации отчетов
   - Создает текстовые отчеты с результатами
   - Сохраняет метрики в CSV
   - Форматирует вывод для презентации

6. main.py
   - ГЛАВНЫЙ СКРИПТ ДЛЯ ЗАПУСКА
   - Выполняет все этапы работы:
     1. Загрузка данных
     2. Обучение моделей классификации
     3. Обучение моделей регрессии
     4. Получение объяснений для классификации
     5. Получение объяснений для регрессии
     6. Генерация отчета

ВСПОМОГАТЕЛЬНЫЕ ФАЙЛЫ

КАК ИСПОЛЬЗОВАТЬ ФАЙЛЫ
======================

ЗАПУСК ПРОГРАММЫ:
  Windows: Двойной клик на run.bat
  Или: python main.py

ПРОСМОТР РЕЗУЛЬТАТОВ:
  1. Откройте results/LAB3_REPORT.txt - основной отчет
  2. Просмотрите графики в директории plots/
  3. Изучите метрики в results/model_metrics.csv

НАСТРОЙКА ПАРАМЕТРОВ:
  Откройте config.py и измените нужные параметры

ПОВТОРНЫЙ ЗАПУСК С НОВЫМИ ДАННЫМИ:
  1. Удалите файлы из data/
  2. Запустите main.py

ИСПОЛЬЗОВАНИЕ ОТДЕЛЬНЫХ МОДУЛЕЙ:
  Смотрите примеры в README.md


ВАЖНЫЕ ЗАМЕЧАНИЯ
================

1. При первом запуске программа:
   - Скачает датасет Wine Quality (~300 KB)
   - Обучит 7 моделей
   - Создаст ~50-70 графиков
   - Займет ~10-20 минут

2. При повторном запуске:
   - Данные берутся из локального кеша (data/)
   - Модели обучаются заново
   - Графики перезаписываются

3. Для ускорения SHAP:
   - Уменьшите SHAP_PARAMS['sample_size'] в config.py
   - По умолчанию: 100 примеров

4. Все графики сохраняются в высоком качестве:
   - Формат: PNG
   - DPI: 300
   - Размер: ~12x8 дюймов
